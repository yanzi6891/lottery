# 人员名单导入说明

## Excel模板格式

请按照以下格式准备人员名单Excel文件：

### 必填列

| 姓名 | 工号 | 部门 |
|------|------|------|
| 张三 | EMP001 | 技术部 |
| 李四 | EMP002 | 产品部 |
| 王五 | EMP003 | 设计部 |
| 赵六 | EMP004 | 运营部 |
| ... | ... | ... |

### 字段说明

- **姓名（必填）**
  - 类型：文本
  - 长度：1-50个字符
  - 唯一性：不能重复
  - 示例：张三、李四、王五

- **工号（可选）**
  - 类型：文本
  - 长度：1-20个字符
  - 示例：EMP001、001、A001

- **部门（可选）**
  - 类型：文本
  - 长度：1-50个字符
  - 示例：技术部、产品部、设计部

### 注意事项

1. **文件格式**
   - 支持：`.xlsx` 和 `.xls` 格式
   - 推荐：使用 `.xlsx` 格式（Excel 2007+）

2. **工作表要求**
   - 只读取第一个工作表（Sheet1）
   - 第一行必须是表头
   - 数据从第二行开始

3. **数据要求**
   - 姓名不能为空
   - 姓名不能重复
   - 每行代表一个人员
   - 建议删除空行

4. **字符限制**
   - 避免使用特殊字符：`* ? / \ [ ] : ; | = , < >`
   - 建议使用中文或英文

### 导入步骤

1. **准备Excel文件**
   - 按照模板格式填写人员信息
   - 检查数据完整性和准确性
   - 保存文件（推荐文件名：参会人员名单.xlsx）

2. **登录管理后台**
   - 浏览器访问：http://localhost:8080/admin
   - 进入"人员管理"页面

3. **上传文件**
   - 点击"导入Excel"按钮
   - 选择准备好的Excel文件
   - 点击"确定"开始导入

4. **验证结果**
   - 查看导入成功数量
   - 检查是否有导入失败的记录
   - 如有错误，根据提示修改后重新导入

### 常见错误

#### 错误1：姓名重复
```
错误：第10行，姓名重复：张三
解决：检查并删除重复的人员
```

#### 错误2：姓名为空
```
错误：第15行，姓名不能为空
解决：填写姓名或删除该行
```

#### 错误3：文件格式错误
```
错误：不支持的文件格式
解决：确保文件是.xlsx或.xls格式
```

#### 错误4：工作表为空
```
错误：工作表中没有数据
解决：检查第一个工作表是否有数据
```

### 数据检查清单

导入前请确认：
- [ ] 文件格式为 `.xlsx` 或 `.xls`
- [ ] 第一行是表头（姓名、工号、部门）
- [ ] 所有姓名已填写且不重复
- [ ] 数据总数为80人
- [ ] 没有空行
- [ ] 没有特殊字符
- [ ] 文件可以正常打开

### 批量操作建议

如果人员较多（80人），建议：
1. 使用Excel的"冻结首行"功能，便于查看表头
2. 使用"数据验证"功能，防止输入错误
3. 导入前先在Excel中排序，便于检查重复
4. 保留一份备份文件
